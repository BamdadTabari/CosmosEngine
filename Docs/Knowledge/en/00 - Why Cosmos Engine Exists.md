# Why Cosmos Engine Exists

---

## Goal

Cosmos Engine is not simply a space simulation engine.

It was created as a long-term learning project to study physics, mathematics, astronomy, cosmology, numerical simulation, and software engineering by implementing scientific concepts rather than only reading about them.

Every line of code should represent a scientific idea, and every new feature should reflect newly acquired knowledge.

---

## Motivation

For me, building has always been a more effective way of learning than memorization.

Instead of only studying scientific theories, I decided to implement them as software.

As the project evolved, its purpose also expanded.

Cosmos Engine became more than a software project; it became a knowledge base documenting both the software and the learning journey behind it.

---

## The Question

If we wanted to recreate the universe inside a computer, where would we begin?

To answer this question we must first understand:

- What is a simulation?
- What is a scientific model?
- How are physical laws described?
- How can mathematics be translated into software?

This handbook attempts to answer these questions step by step.

---

## Intuition

A computer cannot contain the real universe.

However, it can contain a model of the universe.

A model is a simplified representation of reality that preserves only the properties necessary to answer specific questions.

Every scientific simulation is built upon this idea.

---

## Explanation

Cosmos Engine is a physics-based simulation engine.

Its initial goal is to reproduce known physical behavior using well-established scientific models.

Its long-term architecture, however, is intended to support experimentation with alternative physical laws and hypothetical universes.

The objective is not to prove new theories about reality, but to create a platform for learning, experimentation, and scientific exploration.

---

## Scientific View

Science advances through observation, modeling, prediction, and experimentation.

Scientific models are not reality itself; they are our best current descriptions of reality.

Consequently, every simulation is only as accurate as the model on which it is based.

Cosmos Engine follows this principle throughout its design.

---

## Mathematics

Mathematics is the language of simulation.

Vectors, differential equations, geometry, linear algebra, and numerical methods allow physical laws to be transformed into computable algorithms.

These subjects will be introduced progressively throughout this handbook.

## Summary

Cosmos Engine is more than a software project.

It is a lifelong journey toward understanding the universe through simulation, beginning with established physics and expanding through continuous learning.
